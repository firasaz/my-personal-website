Unordered Lists

unordered lists are bullet-point elements of a list

for example

  • piris is savage!
  • piris has the best koo in the world!
  • piris gets random thoughts all the time.

Ordered Lists

ordered lists give some form of order to the elements of a list

for example

  1. i want to drink milk and watch a podcast in the morning when i wake up
  2. then i want to get dressed look nice
  3. lastly but most importantly, conquer the world!

Descriptive Lists

this kind of list doesn't give any number or bullet-point to its title. instead it just indents the contents of its title and leaves it as a description to it

for example

HTML
I describe HTML as the paper that allows us to write and display stuff on.
CSS
I describe CSS as the different pens and colors we use to write and organize and give style with to our paper.
JavaScript
I cannot compare this with an actual paper since we don't interact with the paper whereas in a website we do. In short, JavaScript gives functionality to a webpage on the browser.
